
KA Lite Printed Docs
====================

A repo that combines the PDFs of all KA Lite docs for preparing printable version.

Install
-------

    npm install -g github-wikito-converter



Download
--------

### Latest docs

    wget http://readthedocs.org/projects/ka-lite/downloads/pdf/latest/ -O /chapters/ka-lite-docs.pdf

### KA-Lite wiki

    git clone https://github.com/learningequality/ka-lite.wiki.git
    cd ka-lite.wiki/
    gwtc .
    open documentation.html
    # print with Google Chrome (zoom 130%)




Create LULU project
-------------------

1. Goto lulu.com
2. Now softcover book (8.5" by 11" B&W softcover)
3. Upload content (four PDF files in `chapters/`)
4. Add cover (use images from `covers/`)
5. Order a print copy
